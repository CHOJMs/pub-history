팝업 버튼

공동 인증서 가져오기 (PC → 모바일 앱)

스마트폰에 인증서가 없는 분들은 아래와 같은 방법으로 인증서를 이동한 후 사용바랍니다.

인증서 관리 및 이동을 위해서는 PC에 발급받은 공동인증서가 미리 저장되어 있어야 합니다.

인증서 이동 방법(PC → 모바일 앱)

  • 1
    앱의 ‘공동 인증서 가져오기 (PC → 모바일 앱)’ 메뉴에서 ‘공동 인증서 가져오기’ 실행
  • 2
    PC웹의 메뉴에서 ‘공동인증서 내보내기 (PC → 모바일 앱)‘ 실행
  • 3
    화면에서 ‘인증서 내보내기(PC → 스마트폰)’ 버튼 클릭
  • 4
    인증서 선택 후 비밀번호 입력
  • 5
    스마트폰 화면의 인증번호 입력
  • 6
    다음 버튼 클릭
  • 7
    스마트폰에 인증서 저장완료
