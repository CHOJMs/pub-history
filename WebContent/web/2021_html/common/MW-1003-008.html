팝업 버튼

Face ID (재)등록

사용중인 스마트폰에 등록된 Face ID로
로그인할 수 있는 서비스입니다.

이용안내

  • iM캐피탈은 Face ID 정보를 저장하지 않습니다.
  • Face ID를 이용하기 위해서는 Face ID를 등록 및 인식가능한 스마트폰에서 이용이 가능합니다.
  • Face ID를 이용하기 위해서는 스마트폰에 미리 Face ID 등록이 되어있어야 합니다.
  • iM캐피탈 앱을 삭제하거나 스마트폰을 변경하실 경우 재등록 하셔야 합니다.
  • 연속으로 5회 인증 실패시 Face ID를 재등록 하셔야 합니다.
