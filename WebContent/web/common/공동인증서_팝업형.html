공동인증서

  • 공동인증서 복사
    (PC → 스마트폰)
  • 공동인증서 복사
    (스마트폰 → PC)

  • 1iM캐피탈 앱 실행 후 인증센터 > '공동인증서 복사(PC → 스마트폰)' 메뉴 클릭

  • 2스마트폰에 성생된 인증번호 확인

  • 3하단 '공동인증서 복사(PC → 스마트폰)' 버튼 클릭 > 인증서 선택 후 비밀번호 입력

  • 4스마트폰의 인증번호 입력 후 확인

  • 5스마트폰 하단 확인 버튼 클릭

  • 공동인증서 가져오기 기능을 위해서는 공동인증서가 저장된 PC가 준비되어야 합니다.

  • 1하단 '공동인증서 복사(스마트폰 → PC)' 버튼 클릭

  • 2PC 화면의 인증번호 확인

  • 3iM캐피탈 앱 실행 후 인증센터 > '공동인증서 복사(스마트폰 → PC)' 메뉴 클릭

  • 4복사할 인증서 선택 후 비밀번호 입력

  • 5PC에서 생성된 인증번호 입력 후 확인

  • iM캐피탈 앱에 공동인증서가 저장되어 있어야 합니다.
